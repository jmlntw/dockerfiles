# ECLint Docker Image

A Docker image to run [**ECLint**](https://github.com/jedmao/eclint), an [editorconfig](https://editorconfig.org/) linter.

---

**Usage**

```bash
docker run --rm -v "$(pwd):/usr/src/app" jmlntw/eclint <commands> [files...] [options]
```
